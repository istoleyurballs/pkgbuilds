# Personal PKGBUILD repo

Packages in this repo:
- `archpatch`: Packaging for https://github.com/istoleyurballs/archpatch.
- `grub-theme-minegrub-world-selection-git`: Packaging for https://github.com/Lxtharia/minegrub-world-sel-theme.
- `system-base`: Common packages/config for all my installations.
- `system-ilum`: Config for my desktop.
- `system-install-scripts`: An helper script to automatically install this setup on a machine. See the [INSTALL GUIDE](./INSTALL_GUIDE.md) for usage.
